# 🐱 Благотворительный фонд поддержки котиков QRKot

API-сервис для управления благотворительными проектами и пожертвованиями.

## 🚀 Функционал
|  Модуль | Возможности |
|---------|-------------|
| Проекты | Создание, просмотр, обновление и удаление благотворительных проектов.Автоматическое закрытие при полном финансировании. |
| Пожертвования | Авторизованные пользователи могут делать взносы. Система автоматически распределяет средства между проектами (от старых к новым). |
| Отчёты | Генерация аналитики в Google Sheets: топ проектов по скорости сбора средств, доступ только суперпользователям. |
| Авторизация | JWT-токены, регистрация, управление профилями, роли (пользователь / суперпользователь). |
| Валидация | Строгая проверка данных через Pydantic: уникальные имена проектов, корректные суммы, ограничения на изменение закрытых проектов. |
## 🛠 Технологии
| Категория | Инструменты |
|-----------|-------------|
| Backend | FastAPI, Uvicorn, Pydantic v2 |
| База данных | SQLAlchemy 2.0 (Async), SQLite (aiosqlite), Alembic (миграции) |
| Аутентификация | FastAPI Users, PyJWT |
| Интеграции | Google Sheets API, Google Drive API (через requests и JWT) |
| Инструменты | python-dotenv, flake8, isort |

## 📦 Установка

```bash
# Клонировать репозиторий
git clone <https://github.com/GoodWinee/QRKot>
cd qrkot

# Создать виртуальное окружение
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/macOS
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Настройка окружения - копируйте файл:
cp .env.example .env
# Откройте .env и замените заглушки на реальные данные из вашего Google Cloud проекта.

# Применить миграции
alembic upgrade head

# Запустить сервер
uvicorn app.main:app --reload
```

Приложение будет доступно по адресу: http://127.0.0.1:8000
📖 Swagger документация: http://127.0.0.1:8000/docs
📖 ReDoc: http://127.0.0.1:8000/redoc
